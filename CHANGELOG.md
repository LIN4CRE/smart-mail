# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-07-07

### Added
- Complete modern unified inbox experience.
- AI (Gemini) powered smart categorization, summarization, and action extraction.
- Gmail integration with bulk archive, trash, mark as read, and unsubscribe features.
- PWA support with auto-updating capabilities.
- Dark mode terminal/cyberpunk-influenced responsive interface.
